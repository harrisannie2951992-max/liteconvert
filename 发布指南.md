# 🚀 发布指南 — 三步把「轻转」开源给全世界用

你现在拿到的这个文件夹就是完整的开源项目。跟着下面走,大约 10 分钟,
结束后你会得到一个 **任何人打开网址就能用** 的在线版本。

---

## 第 1 步 · 注册 GitHub 账号(约 2 分钟)

1. 打开 <https://github.com/signup>
2. 填邮箱 → 设置密码 → 取一个用户名(比如 `claudine-spencer`,后面网址会用到)
3. 完成邮箱验证即可,免费账号就够用

## 第 2 步 · 创建仓库并上传(约 5 分钟,全程网页操作,不用命令行)

1. 登录后点右上角 **+** → **New repository**
2. Repository name 填 `liteconvert`,选 **Public**,直接点 **Create repository**
3. 在新仓库页面点 **uploading an existing file** 链接
4. 打开你 Mac 上的「轻转 LiteConvert」文件夹,**全选里面所有文件和文件夹**
   (`index.html`、`assets`、`vendor`、`icons` 等),拖进网页上传区
   > 注意:是拖文件夹里的**内容**,不是拖整个外层文件夹
5. 等待上传完成,底部点 **Commit changes**

> 会用命令行的话更快:
> ```bash
> cd 轻转LiteConvert所在目录
> git init && git add -A && git commit -m "轻转 LiteConvert v1.0"
> git branch -M main
> git remote add origin https://github.com/<你的用户名>/liteconvert.git
> git push -u origin main
> ```

## 第 3 步 · 开启 GitHub Pages,获得公开网址(约 2 分钟)

1. 仓库页面 → **Settings** → 左侧 **Pages**
2. 「Build and deployment」下的 Branch 选择 **main**,文件夹保持 **/(root)**,点 **Save**
3. 等 1–3 分钟,刷新页面,顶部会显示你的网址:
   **`https://<你的用户名>.github.io/liteconvert/`**

把这个网址发给任何人,对方打开就能用;用 Safari 打开还能
「添加到程序坞 / 主屏幕」当 App 使。HTTPS 环境下离线缓存会自动生效。

---

## 发布后的小优化(可选)

- **替换 GitHub 链接**:编辑 `index.html`,把两处 `https://github.com/`
  改成你的仓库地址 `https://github.com/<你的用户名>/liteconvert`
- **加个截图**:把 `test/out/ui_light.png` 传到仓库,在 README 里引用,更吸引人
- **删掉测试目录**:`test/` 文件夹只用于开发,发布时可以不上传
- **收集反馈**:仓库 Settings → Features 里开启 Issues,用户就能给你提建议

## 以后想更新怎么办?

改完文件后,在仓库页面点 **Add file → Upload files** 重新上传同名文件覆盖即可,
GitHub Pages 会在几分钟内自动更新。

有任何一步卡住了,回来告诉我,我可以陪你一步步操作(也可以帮你直接在浏览器里点)。
